== Description ==

Displays a message above post content warning visitors of potential hazards of reading your content


== Installation ==


1. Upload `/iwarnedyou/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I change the warning message? =
Post Menu > I Warned You
